## 集成点

### 1. 视觉处理管道

```
原始视频文件
    ↓ fetch_video_raw_with_scenes()
原始张量 + 场景边界（原始帧索引）
    ↓ resample_video_from_raw()
处理后的张量
    ↓ map_raw_scenes_to_processed()
场景边界（处理后的帧索引）
    ↓
数据集/训练
```

### 2. RL数据集管道

```
数据框行
    ↓
MultiModalDatasetVideo.__getitem__()
    ├─ 检查 use_scene_detection 标志
    ├─ 如果为 True: 调用 process_video_adaptive_token_num_with_scenes()
    ├─ 否则: 调用 process_video_adaptive_token_num()
    └─ 将 scene_boundaries 存储在输出中
    ↓
训练循环
    └─ 通过 sample['scene_boundaries'] 访问
```

### 3. SFT数据集管道

```
视频文件
    ↓
_regularize_videos_av() 或 _regularize_videos_decord()
    ├─ 检查 USE_SCENE_DETECTION 环境变量
    ├─ 如果为真: 从加载的帧检测场景
    └─ 如果检测到则包含在输出中
    ↓
SFT训练
    └─ 场景边界可用于使用
```
